# online-retail-customer-analysis

## Overview
This project analyzes customer purchasing behavior using Python and Power BI.

The analysis focuses on:
- VIP customer segmentation
- Churn risk analysis
- Cohort retention analysis
- Return risk analysis
- Customer purchasing behavior patterns

---

## Key Insights
- VIP customers contributed a disproportionately large share of total revenue.
- VIP revenue growth was driven more by purchase frequency and quantity rather than higher product prices.
- Churn risk customers showed declining engagement patterns before inactivity.
- Some VIP customers exhibited slowing purchase behavior, indicating potential early churn signals.
- High return-rate VIP customers were identified as potential profitability risks.
